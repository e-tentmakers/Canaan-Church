# 가나안교회 웹사이트

가나안교회 공식 웹사이트입니다.

## 프로젝트 구조

```
Canaan Church/
├── index.html              # 메인 페이지
├── css/
│   └── style.css          # 스타일시트
├── js/
│   └── script.js          # JavaScript 파일
├── image files/           # 이미지 파일 디렉토리
└── pages/
    ├── about/             # 가나안교회 메뉴
    │   ├── intro.html     # 교회 소개
    │   ├── greeting.html  # 인사말
    │   ├── staff.html     # 섬기는 사람
    │   └── location.html  # 오시는 길
    ├── worship/           # 예배 메뉴
    │   ├── sunday.html    # 주일예배
    │   ├── wednesday.html # 수요기도회
    │   └── dawn.html      # 새벽기도회
    ├── board/             # 게시판 메뉴
    │   ├── column.html    # 목회 칼럼
    │   └── bulletin.html  # 게시판
    └── gallery/           # 갤러리 메뉴
        ├── 2024.html      # 2024년 사역
        └── 2025.html      # 2025년 사역
```

## 기능

- 반응형 디자인 (PC 및 모바일 지원)
- 드롭다운 네비게이션 메뉴
- 모바일 햄버거 메뉴
- 모든 페이지에 일관된 디자인

## 이미지 파일

다음 이미지 파일을 `image files/` 디렉토리에 배치해야 합니다:

- `가나안교회 2025 예배.png` (메인 페이지 두 번째 섹션에 사용)
- `교회비전 Image 3.jpg` (필요한 페이지에 사용)

## 로컬에서 실행

1. `index.html` 파일을 웹 브라우저에서 엽니다.
2. 또는 로컬 웹 서버를 사용하여 실행할 수 있습니다.

## GitHub Pages 배포

GitHub 저장소에 푸시한 후, GitHub Pages를 활성화하여 웹사이트를 배포할 수 있습니다.
